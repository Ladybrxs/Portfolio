This interactive Power BI dashboard is designed to provide a comprehensive view of a retailer's performance, allowing users to identify sales trends, manage inventories and monitor out-of-stocks. Calculations in DAX are used to create metrics and calculated columns that facilitate deeper analysis of the data.

Contents
Dashboard in Power BI: Interactive visualizations of sales and inventory.
DAX Calculations: Includes calculated measures and columns for detailed analysis.
Time Trends: Analysis of how sales and inventory vary over time.

Technologies Used
Power BI
DAX (Data Analysis Expressions)
