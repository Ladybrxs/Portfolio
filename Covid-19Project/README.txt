This project uses SQL to analyze data on COVID-19, exploring multiple tables to extract information on cases, deaths and recovery. It is complemented by a dashboard in Tableau that presents interactive visualizations, allowing a clear analysis of the evolution of the pandemic and its impacts.

Contents
SQL queries: Code for COVID-19 data extraction.
Tableau Dashboard: Interactive visualizations showing trends and key statistics.

Technologies Used
SQL
Tableau
