This project analyzes Starbucks data through SQL, using multiple tables to extract relevant information about sales, products and customers. The analysis is complemented with a Tableau dashboard that visualizes the results in an intuitive way, facilitating the understanding of patterns and trends in the data.

Contents
SQL queries: Code for data exploration across multiple tables.
Dashboard in Tableau: Interactive visualizations that show insights from the analyzed data.

Technologies Used
SQL
Tableau
