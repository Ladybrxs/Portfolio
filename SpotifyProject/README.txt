This project analyzes Spotify data using SQL queries for exploratory analysis and an interactive dashboard in Power BI, allowing users to explore listening trends, music genres, and other relevant patterns. The dashboard provides intuitive visualizations that make it easy to understand how users interact with music on the platform.

Contents
Dashboard in Power BI: Interactive visualizations of Spotify data.
SQL Data Analysis: Exploratory queries to identify trends and listening patterns.

Technologies Used
Power BI
SQL
